## Как начать использовать
1. Установка пакета кита в проект
```
npm i @wb-tech/ui-react -S
```

2. Установка пакета билдера в проект. (В примере используется вариант для webpack 5)
- [Builder webpack 5](?path=/docs/builders-webpack-5--документация)
```
npm i @wb-tech/builder-webpack-5 -D
```

3. Подключение билдера ([Документация по @wb-tech/builder-webpack-5](?path=/docs/builders-webpack-5--документация))

4. Указать область действия дизайн токенов
```tsx
import React from 'react';
import { render } from 'react-dom';
import { DesignTokens } from '@wb-tech/ui-react';

// ...

render((
    <Provider store={store}>
        <DesignTokens>
            <Routes />
        </DesignTokens>
    </Provider>
));
```

5. Использовать компоненты `@wb-tech/ui-react`
